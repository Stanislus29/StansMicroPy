# Examples

MicroPython example scripts organized by component. Each subfolder targets a specific hardware component with progressively complex examples.

## Folder Structure

```css
Examples/
├── Button/          # Push button examples
├── LCD/             # I2C LCD display examples
├── LED/             # LED blink, brightness, and multi-LED examples
├── OLED/            # SSD1306 OLED display examples
├── RGBLED/          # RGB LED color mixing examples
├── Servo/           # Servo motor control examples
├── Ultrasonic/      # HC-SR04 distance sensor examples
└── Integrated/      # Multi-component projects (sound + servo, ultrasonic + LED, etc.)
```
