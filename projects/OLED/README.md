# OLED Examples

Examples for using an SSD1306 OLED display with the Raspberry Pi Pico over I2C.

## Folder Structure

```
OLED/
├── oled_init_test.py             # Initialize the SSD1306 OLED (basic connection test)
├── oled_temperature_display.py   # Read Pico's internal temperature, display on OLED
└── ssd1306.py                    # SSD1306 driver (local copy for standalone use)
```
