# Servo Examples

Examples for controlling servo motors with the Raspberry Pi Pico using PWM.

## Folder Structure

```
Servo/
├── servo_move_to_angle.py      # Move servo to 90° then 0° (Servo library)
├── servo_sweep.py              # Sweep servo 0°–180° in 2° steps (Servo library)
├── servo_oscillate.py          # Oscillate servo between 30° and 150° (Servo library)
└── servo_sweep_baremetal.py    # Sweep servo using bare machine.PWM (no library)
```
