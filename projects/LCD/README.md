# LCD Examples

Examples for using an I2C LCD display with the Raspberry Pi Pico.

## Folder Structure

```
LCD/
├── i2c_address_scan.py   # Scan the I2C bus and detect connected device addresses
├── lcd_hello_world.py    # Display "Hello World" on a 16x2 I2C LCD
└── lcd_two_lines.py      # Display text on both lines of a 16x2 I2C LCD
```
