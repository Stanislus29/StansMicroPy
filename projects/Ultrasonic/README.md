# Ultrasonic Sensor Examples

Examples for using the HC-SR04 ultrasonic distance sensor with the Raspberry Pi Pico.

## Folder Structure

```
Ultrasonic/
├── ultrasonic_measure_distance.py        # Continuously measure and print distance (cm)
├── ultrasonic_led_servo.py               # LED + servo react when distance > 50 cm
└── ultrasonic_led_servo_threshold.py     # LED + servo react when distance > 20 cm
```
