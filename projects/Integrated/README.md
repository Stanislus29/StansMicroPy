# Integrated Examples

Multi-component projects that combine several sensors and actuators together.

## Folder Structure

```
Integrated/
├── keypad_read.py               # Read pressed keys from a 4x4 matrix keypad
├── sound_sensor_led.py          # Turn on LED when sound exceeds threshold
├── sound_sensor_servo.py        # Sweep servo when loud sound is detected
├── ultrasonic_led_baremetal.py   # LED reacts to distance (bare machine.Pin, no library)
└── leds_and_servo.py            # Blink LEDs and oscillate servo simultaneously
```
